# 🏏 Cricket Analysis Dashboard

## 📌 Project Overview

The Cricket Analysis Dashboard is an interactive Power BI project designed to analyze player performance across Batting, Bowling, and Fielding metrics. The dashboard provides meaningful insights through visualizations, KPIs, and filters to help users evaluate player performance effectively.

## 🎯 Objectives

- Analyze batting performance statistics
- Evaluate bowling effectiveness
- Measure fielding contributions
- Provide interactive filtering and drill-down capabilities
- Generate actionable insights from cricket data


## 📊 Dashboard Pages

### 🏏 Batting Analysis
- Total Runs
- Batting Average
- Strike Rate
- Boundaries Analysis
- Player-wise Performance

### 🎯 Bowling Analysis
- Total Wickets
- Economy Rate
- Bowling Average
- Overs Bowled
- Bowling Performance Comparison

### 🧤 Fielding Analysis
- Catches
- Run Outs
- Stumpings
- Fielding Contribution Analysis

## 🛠️ Technologies Used

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization
